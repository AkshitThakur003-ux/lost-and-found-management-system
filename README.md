---------------------------------Lost & Found Management System------------------------------------------

A modular C++ console project demonstrating OOP, Templates, Exception Handling, and File Storage.

 ---> Overview

The Lost & Found Management System is a console-based C++ application that helps users record and manage lost or found items.
The project demonstrates Object-Oriented Programming (OOP), file handling, templates, exception handling, and clean modular design.
All data is stored in a simple text file so it remains available across program runs.

----> Features

 - Register Lost Item

 - Register Found Item

 - Display All Items (Formatted Table)

 - Search Items by Name

 - Update Item by ID

 - Delete Item by ID

 - Help & About Section

 - Persistent File Storage (data.txt)

 - Robust Input Validation

 - Supports Serialization & Deserialization

 - Demonstrates Core OOP Concepts:

 - Abstraction

 - Polymorphism

 - Inheritance

 - Encapsulation

 - Templates
